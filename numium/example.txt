2. Time lib:
this lib can get time now and can delay your program 

syntaxes:
getTimeNow(<you can request to get which get time in which country (use utc)>)
delay(<count>)
// note: use <
  %H: hour
  %MI: min
  %S: second
  %D: Day
  %MO: month
  %Y: year
> (use in getTimeNow)

```
import <numium_stdio>
import <environment>
import <time>

INIT environment <env>
env START 

area module main() open
    env time string = getTimeNow("UTC+7", "%Y:%MO:%D - %H:%MI:%S)
    // getTimeNow(country type utc, type)
    output(time)
close 
```

3. kernel_linker
this is lib run commands
only has: system() and pwd()

```
import <numium_stdio>
import <environment>
import <time>
import <kernel_linker>

INIT environment <env>
env START

area module main() open
    env user string = system("whoami") // it will give back if is command back, or it will run without back
    env path string = pwd()
    output(user)
    output(path)
close 

```
